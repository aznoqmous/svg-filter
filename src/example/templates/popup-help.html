Quick start

Start by dragging the Source Graphic or Source Alpha output to add your first filter, then repeat !
Note that your filter wont be completed until you link an output to the Result block.

You can click on any input to unlink it.

Shortcuts

  • Zoom : Mouse wheel
  • Move : Hold Space then use your mouse
  • Backspace or Delete : Remove selected filters
  • Right Click : Open the add filter menu
